Selctors in css

Element selector

Element selector is used to select all the elements of a specific type. For example, if you want to select all the paragraphs in a document, you can use the element selector "p".

Id selector

Id selector is used to select a specific element with a unique id. The id selector is defined by using the "#" symbol followed by the id of the element. For example, if you want to select an element with the id "header", you can use the id selector "#header".

Class selector

Class selector is used to select all elements with a specific class. The class selector is defined by using the "." symbol followed by the class name. For example, if you want to select all elements with the class "highlight", you can use the class selector ".highlight".

Universal selector

Universal selector is used to select all elements in a document. The universal selector is defined by using the "*" symbol. For example, if you want to apply a style to all elements in a document, you can use the universal selector "*".
